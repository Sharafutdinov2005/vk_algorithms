# Стабильность

Порядок ненулевых элементов останется неизменным в силу последовательного движения `ptr`, аналогично задаче с чётными числами.

# Временная сложность

Рассмотрим код функции `move_zeros`:

```python
ptr_positive, ptr = 0, 0

    while ptr < len(array):
        if array[ptr] != 0:
            array[ptr_positive], array[ptr] = array[ptr], array[ptr_positive]
            ptr_positive += 1
        ptr += 1
```

Вычислим количество операций, выполняемых фукцией:
- до цикла функция выполняет постоянное количество операций:
$$
C_1
$$
- В цикле указатель `ptr` движутся от начала и конца массива, производя N операций:
$$
N * C_2 + C_1
$$

Таким образом итоговая асимптотика функции при больших N:
$$
O(N)
$$
