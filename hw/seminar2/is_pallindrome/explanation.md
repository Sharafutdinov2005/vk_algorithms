# Сравнение символов

Два указателя симметрично движутся от начала и конца списка, проводя сравнение аналогичных элементов.

# Временная сложность

Рассмотрим код функции `is_pallindrome`:

```python
    left, right = 0, len(string) - 1

    while left < right:
        if string[left] != string[right]:
            return False
        left += 1
        right -= 1

    return True
```

Вычислим количество операций, выполняемых фукцией:
- до цикла функция выполняет постоянное количество операций:
$$
C_1
$$
- В цикле указатели движутся от начала и конца массива на встречу друг другу, производя в худшем случае $[N / 2]$ операций :
$$
N / 2 * C_2 + C_1
$$

Таким образом итоговая асимптотика функции при больших N:
$$
O(N)
$$
