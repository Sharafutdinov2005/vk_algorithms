# Обределение исходности строки

Наиболее наглядно решение работает при использовании стека. Мы знаем, что если строка `a` является исходной для `b`, то `b` получена из `a` добавлением элементов. Пойдём по строке `b` и при псоледовательном появлении элемента из `a` будем их отбрасывать. Если элементы в `a` кончатся - она будет исходной. Иначе - не будет.

При использовании указателей мы не отбрасываем элементы `a`, а просто сдвигаем указатель на следующий элемент. Нужно только добавить защиту от выхода за пределы списка.
